## Tech Stack:
- Node.js
- Hapi
- Cloud SQL (MySQL)
- App Engine

## How to run:
1. npm install
2. npm run start

## How to deploy:
1. git clone https://github.com/C23-PR539-NutriPlan/backend.git
2. run "gcloud app deploy" on cloud shell in the backend directory
